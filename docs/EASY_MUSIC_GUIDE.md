## 🎯 このアプリでできること

### AIが音楽を自動生成！

このアプリは **ACE-Step** というAI音楽生成エンジンを使って、テキストから音楽を生成します。

- 🎤 **AI作詞**: テーマを入力するだけで歌詞を自動生成
- 🏷️ **タグ生成**: ジャンルや楽器を自動で提案
- 🎵 **音楽生成**: 歌詞とタグから実際の音楽を生成
- 🎧 **ビジュアライザー**: 音楽に合わせたカラフルなアニメーション

### こんな人におすすめ

- 🎹 楽器が弾けなくても曲を作りたい
- 📝 作詞のアイデアが欲しい
- 🎬 動画のBGMが必要
- 🎮 ゲームや創作活動に音楽を使いたい

## 📚 音楽用語かんたん解説

### 🎵 基本用語

| 用語 | 読み方 | 意味 | 例 |
|------|--------|------|-----|
| **BPM** | ビーピーエム | 1分間の拍数（テンポの速さ） | 60=ゆっくり、120=普通、180=速い |
| **調（キー）** | ちょう（キー） | 曲の基準となる音階 | C major=明るい、A minor=暗い |
| **拍子** | ひょうし | リズムの基本単位 | 4/4=一般的、3/4=ワルツ |

### 🎸 ジャンル用語

| ジャンル | 特徴 | BPM目安 |
|---------|------|---------|
| **Pop** | 親しみやすいメロディ | 100-130 |
| **Rock** | 力強いギターサウンド | 110-140 |
| **EDM** | 電子音楽、クラブ向け | 120-150 |
| **Ballad** | ゆったりした感動的な曲 | 60-80 |
| **Jazz** | 洗練されたアドリブ感 | 80-120 |
| **Hip Hop** | ラップ向けビート | 80-100 |
| **Classical** | オーケストラ的 | 60-120 |
| **Ambient** | 環境音楽、リラックス | 60-90 |

### 🎹 楽器タグの例

| カテゴリ | 楽器例 |
|---------|--------|
| **弦楽器** | guitar, violin, bass, cello |
| **鍵盤** | piano, keyboard, synthesizer, organ |
| **打楽器** | drums, percussion, beatbox |
| **管楽器** | saxophone, trumpet, flute |
| **電子** | synth, electronic, 808 |
| **声** | vocal, choir, harmony |

### 🎼 調（キー）の選び方

#### メジャーキー（明るい曲向け）
- **C major**: 最も基本的、明るく素直
- **G major**: 爽やか、ポップ向け
- **D major**: 明るく華やか
- **F major**: 温かみがある

#### マイナーキー（暗い・切ない曲向け）
- **A minor**: 最も基本的なマイナー
- **E minor**: 哀愁、ロック向け
- **D minor**: ドラマチック
- **B minor**: 深い悲しみ

> 💡 **迷ったら**: 明るい曲なら「C major」、暗い曲なら「A minor」を選べばOK！

## ⚡ 5分で作る！はじめての曲

### ステップ1: テーマを入力（30秒）

「テーマ・シナリオ」欄に、作りたい曲のイメージを書きます。

**入力例:**
```
夏の海辺で恋人と過ごす思い出の曲。夕日が沈む美しい風景。
```

### ステップ2: AI作詞（30秒）

1. 「🤖 AI作詞」ボタンをクリック
2. AIが歌詞を自動生成
3. 気に入らなければもう一度クリック

### ステップ3: タグ生成（30秒）

1. 「🏷️ タグ生成」ボタンをクリック
2. ジャンルと楽器が自動で提案される
3. 必要に応じて編集

### ステップ4: パラメータ設定（1分）

基本的にはデフォルトでOK！こだわりたい場合：

| 設定 | おすすめ |
|------|---------|
| 長さ | 30-60秒（お試し用）、120-180秒（本格的） |
| BPM | ポップなら120、バラードなら70 |
| 調 | 明るい曲→C major、暗い曲→A minor |

### ステップ5: 音楽生成（2-3分）

1. 「🎵 音楽を生成」ボタンをクリック
2. 生成完了を待つ（約1-3分）
3. 完成！再生ボタンで聴いてみよう

## 🎨 もっと良い曲を作るコツ

### 💡 テーマの書き方

#### ❌ 悪い例
```
楽しい曲
```

#### ✅ 良い例
```
春の桜並木を歩く若いカップル。
希望に満ちた新生活のスタート。
明るく爽やかなポップソング。
```

**ポイント:**
- 場面や情景を具体的に
- 感情や雰囲気を言葉にする
- ジャンルの希望も書く

### 🏷️ タグのカスタマイズ

**ジャンルタグの組み合わせ例:**
```
pop, acoustic, romantic, emotional
```
```
rock, energetic, electric guitar, powerful drums
```
```
lofi, hip hop, chill, relaxing, piano
```

**楽器の追加で個性を:**
- `strings` - ストリングスで壮大に
- `acoustic guitar` - 温かみのある音
- `synthesizer` - 近未来的なサウンド
- `piano` - クラシカルな雰囲気

### ⚙️ パラメータ調整

#### STEP（推論ステップ数）
| 値 | 効果 | 生成時間 |
|----|------|---------|
| 30-50 | 標準品質 | 速い |
| 60-100 | 高品質 | 普通 |
| 100-200 | 最高品質 | 遅い |

> 💡 Turboモデルでは自動的に8ステップに最適化されます

#### CFG（ガイダンススケール）
| 値 | 効果 |
|----|------|
| 1.0-2.0 | 自由度高い、予想外の展開 |
| 3.0 | バランス良い（おすすめ） |
| 5.0-10.0 | タグに忠実、安定 |

#### SEED（シード値）
- 同じ設定 + 同じSEED = 同じ曲が生成
- お気に入りの曲ができたらSEEDをメモ！
- 「ランダム」のままなら毎回違う曲

### 🎯 ジャンル別おすすめ設定

#### ポップソング
```
BPM: 120
調: C major または G major
タグ: pop, catchy, vocal, drums, bass, keyboard
長さ: 180秒
```

#### バラード
```
BPM: 70
調: A minor または F major
タグ: ballad, emotional, piano, strings, vocal
長さ: 240秒
```

#### EDM/ダンスミュージック
```
BPM: 128
調: A minor
タグ: edm, electronic, synth, bass drop, energetic
長さ: 180秒
```

#### LoFi/作業用BGM
```
BPM: 85
調: D major
タグ: lofi, hip hop, chill, piano, vinyl crackle
長さ: 180秒
```

## ❓ よくある質問

### Q: 生成にどのくらい時間がかかる？

**A:** 設定や混雑状況により異なります：
- 30秒の曲: 約30秒〜1分
- 180秒の曲: 約1〜3分
- STEPを増やすと長くなる

### Q: 生成した曲は自由に使える？

**A:** 個人利用は基本的にOKです。商用利用については、ACE-Stepのライセンスを確認してください。

### Q: 同じ設定で違う曲を作りたい

**A:** SEEDを「ランダム」にしておけば、毎回異なる曲が生成されます。

### Q: 気に入った曲をもう一度作りたい

**A:** 生成時に表示されるSEED値をメモしておき、同じ設定で再生成すれば同じ曲ができます。

### Q: 歌詞を自分で書きたい

**A:** 「歌詞」欄に直接入力できます。AI作詞を使わなくてもOK！

フォーマット例：
```
[verse]
ここに歌詞を書く
1行ずつ書いていく

[chorus]
サビの歌詞
盛り上がる部分

[outro]
終わりの部分
```

### Q: ボーカルなしの曲を作りたい

**A:** タグに `instrumental` を追加し、歌詞欄を空にするか `[instrumental]` とだけ書いてください。

### Q: エラーが出て生成できない

**A:** 以下を確認してください：
1. ACE-Step APIサーバーが起動しているか
2. インターネット接続が安定しているか
3. 歌詞やタグに不正な文字がないか

## 📝 歌詞の構造タグ

歌詞を書くときに使えるタグ：

| タグ | 意味 | 説明 |
|------|------|------|
| `[intro]` | イントロ | 曲の導入部分 |
| `[verse]` | バース/Aメロ | 物語を進める部分 |
| `[pre-chorus]` | プレコーラス/Bメロ | サビ前の盛り上げ |
| `[chorus]` | コーラス/サビ | 曲のメイン、繰り返し |
| `[bridge]` | ブリッジ/Cメロ | 変化をつける部分 |
| `[outro]` | アウトロ | 曲の終わり |
| `[instrumental]` | 間奏 | 歌なしの部分 |

### 歌詞構造の例

```
[intro]
(ピアノのイントロ)

[verse]
朝日が昇る街の中
新しい一日が始まる

[pre-chorus]
心の準備はできている
さあ、踏み出そう

[chorus]
未来へ続く道を歩こう
どんな困難も乗り越えて
夢を叶える日まで

[verse]
...

[chorus]
...

[outro]
(フェードアウト)
```
